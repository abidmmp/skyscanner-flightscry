Android application built with Kotlin and Skyscanner Backpack UI components as part of the Skyscanner Software Engineering Virtual Experience Program on Forage.

## Technologies

- Kotlin
- Android Studio
- Backpack Android
- Gradle
- XML Layouts

## Features

- Flight Information Card
- Departure Card
- Arrival Card
- Backpack UI Components
- Material-inspired Android Interface

## Learning Outcomes

- Android UI development
- Backpack Android components
- Kotlin basics
- XML layouts
- Gradle dependency management
